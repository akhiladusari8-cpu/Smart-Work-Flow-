# 🚀 Smart Workflow Manager
 
Live Link :- https://smart-workflow--akhiladusari8.replit.app

## 📌 Overview
Smart Workflow Manager is a simple and powerful web application designed to reduce **Tool Overload and Switching Cost** by combining multiple productivity tools into one dashboard.

Instead of switching between different apps like notes, tasks, and timers, this app allows users to manage everything in one place — improving focus and efficiency.

---

## 🎯 Problem Statement
Frequent switching between multiple applications leads to:
- Loss of focus
- Reduced productivity
- Increased mental fatigue

This project solves the issue by creating an **all-in-one productivity environment**.

---

## 💡 Features

### ✅ Task Manager
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Data stored using localStorage

### 📝 Notes Section
- Add quick notes
- Delete notes
- Persistent storage

### ⏱ Focus Timer (Pomodoro)
- 25-minute work session
- Start / Pause / Reset functionality
- Helps maintain concentration

### 📊 Activity Tracker
- Tracks completed tasks
- Displays total focus time
- Shows switching count

### 🔁 Switching Cost Tracker (Unique Feature)
- Tracks how often the user switches between sections
- Displays alert when switching too frequently

### 🌙 Dark Mode
- Toggle between light and dark theme
